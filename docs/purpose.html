Our purpose is to generate trust and we do this by (re)generating communities where new stakeholders relationships are created, by proposing cultural and artistic action to stimulate participation, by giving destinations the ownership of the value they generate (e.g. data), by involving destinations in the governance of the global platform, by making data and services accessible through open APIs and by releasing all the code we create as open source software.
